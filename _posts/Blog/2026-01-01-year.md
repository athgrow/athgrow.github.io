# 4. Jekyll 과 Bundler 설치 

> <u>Bundler</u>는 루비 프로젝트에 필요한 gem들의 올바른 버전을 추적하고 설치해서 일관된 환경을 제공하는 도구이다.  

cmd에 다음과 같은 명령어를 수행한다. `gem install jekyll bundler`
cmd에 `jekyll -v` 명령어를 수행하여 jekyll이 잘 설치되었는지 확인해본다.
